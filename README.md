# Battleship-with-Java

## Overview
Text-based Battleship implementing player input validation, board rendering, and fog-of-war. Hyperskill project.

## Tech
- Java 17+

## How to run

```bash
javac -d out src/*.java
java -cp out Main
```

## Features
- Board setup & coordinate parsing
- Turn loop and hit/miss markers
- Error handling for invalid input
